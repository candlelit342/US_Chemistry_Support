Standard Reduction Potentials — Combined Reference Table

All electrode couples used in the galvanic cell simulation (Bratsch, 1989), combined with the full standard reduction potentials table from the IB Chemistry data booklet, sorted together as one electrochemical series.

About this table

Two different literature sources are combined here, and both are kept fully traceable rather than merged into single "averaged" values:

S — used in this simulation, from Bratsch (1989) IB — IB Chemistry data booklet (2025), Section 19

Where a species appears in both sources (e.g. Mg, Al, Mn, Zn, Fe, Ni, Sn, Pb, Cu, Ag), you'll see two adjacent rows with slightly different E° values. This is not an error — it reflects the two sources using different levels of precision and, in places, slightly different underlying measurements. The simulation itself always uses the Bratsch (1989) values internally. The IB data booklet values are what you should quote in IB exam contexts.

Only couples of the form Mn+(aq) + n e⁻ ⇌ M(s) — a metal in equilibrium with its own aqueous ion — can be used directly as one of the two electrodes in this simulator. The other entries from the IB data booklet (involving H₂, O₂, halogens, MnO₄⁻, Cr₂O₇²⁻, Fe³⁺/Fe²⁺, Cu²⁺/Cu⁺, etc.) are not simple solid-metal electrodes and cannot be selected in the simulation, but are included here for a complete reference electrochemical series — useful for predicting the feasibility of any redox reaction, not just the ones this simulator can build.
