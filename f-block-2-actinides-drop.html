f-Block Part 2: The Actinide Series

Fourteen mystery blocks — Th through Lr (atomic numbers 90–103) — will fall one at a time, in random order. Use the arrow keys to steer each block sideways, and land it in its correct box in the Actinide row beneath the main table before it reaches the floor. The full periodic table above (including La* and Ac*, whose footnotes point down to the f-block) and the other f-block series are shown for reference throughout.

Correct placements score more points the faster you find them. Wrong placements cost points and send the block back up for another try.

Use the switches above the board any time — even mid-game — to build your own challenge: hide the category colours for double points, hide the ghost-letter hints for triple points, or tick "Remove all scaffolding! The expert level!" for a flat six times the points (the same effect as ticking both individually, but in one click).
