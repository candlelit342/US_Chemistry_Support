OIL RIG  —  Oxidation Is Loss of electrons  |  Reduction Is Gain of electrons
In terms of oxidation numbers: oxidation = increase in ON  |  reduction = decrease in ON

Key Rules for Oxidation Numbers

Rule 1
Pure elements have ON = 0. (H2, Fe, P4, S8…)
Rule 2
For an ion, the sum of ONs = the ion's charge. (Fe2+ → Fe = +2)
Rule 3
For a neutral molecule, sum of all ONs = 0.
Fluorine
Always −1. No exceptions.
Hydrogen
Usually +1. Exception: metal hydrides (e.g. NaH) → −1.
Oxygen
Usually −2. Exception: peroxides (H2O2) → −1.
Group 1 metals
Always +1 in compounds. (Na, K, Li…)
Group 2 metals
Always +2 in compounds. (Mg, Ca, Ba…)

Worked Solutions

Q8a
Cu2+ → CuI
Cu²⁺: Single ion, charge +2. Cu = +2
CuI: Neutral. I (halide) = −1. Cu + (−1) = 0. Cu = +1
Change: +2 → +1. ON decreased. Cu has been REDUCED.
Q8b
NO2 → N2O4
NO₂: Neutral. O = −2, two O = −4. N + (−4) = 0. N = +4
N₂O₄: Neutral. O = −2, four O = −8. 2N + (−8) = 0. N = +4
Change: +4 → +4. No change. NEITHER. (N₂O₄ is the dimer of NO₂ — dimerisation doesn't change ON.)
Q8c
PH3 → HPO22−
PH₃: Neutral. H = +1, three H = +3. P + 3 = 0. P = −3
HPO₂²⁻: Charge = −2. H = +1, O = −2, two O = −4. P + 1 + (−4) = −2. P = +1
Change: −3 → +1. ON increased. P has been OXIDISED.
Q8d
CH3OH → HCOOH
CH₃OH: Neutral. 3H = +3, O = −2, H on O = +1. C + 3 + (−2) + 1 = 0. C = −2
HCOOH: Neutral. 2H = +2, 2O = −4. C + 2 + (−4) = 0. C = +2
Change: −2 → +2. ON increased. C has been OXIDISED. (This is why converting an alcohol to a carboxylic acid is called an oxidation reaction.)
Q8e
S2O32− → S4O62−
S₂O₃²⁻: Charge = −2. 3O = −6. 2S + (−6) = −2. 2S = +4. S = +2
S₄O₆²⁻: Charge = −2. 6O = −12. 4S + (−12) = −2. 4S = +10. S = +2.5
Change: +2 → +2.5. ON increased. S has been OXIDISED. (Fractional ONs are valid — they represent an average across the sulfur atoms.)
Question 9 · Extended Question

Hydrazine & Dinitrogen Tetroxide

Hydrazine (N2H4) and dinitrogen tetroxide (N2O4) react violently to form nitrogen gas as a product from both reactants. This reaction has been used as rocket fuel.
Question 9a
Explain why this is a redox reaction, stating what is being oxidised and what is being reduced.
Step 1 — Oxidation numbers in the reactants
N₂H₄: Neutral. H = +1, four H = +4. 2N = −4. N = −2
N₂O₄: Neutral. O = −2, four O = −8. 2N = +8. N = +4
N₂: Elemental. N = 0
Step 2 — Identify changes
N in N₂H₄: −2 → 0. ON increased. N₂H₄ is OXIDISED.
N in N₂O₄: +4 → 0. ON decreased. N₂O₄ is REDUCED.
Since one species is oxidised and another is reduced, this is a redox reaction.
Question 9b
Write a balanced equation for this reaction.
Balanced equation
2N2H4 + N2O4 → 3N2 + 4H2O
Atom check
N: 4 + 2 = 6 left · 3×2 = 6 right ✓
H: 2×4 = 8 left · 4×2 = 8 right ✓
O: 4 left · 4×1 = 4 right ✓
Electron check: N in N₂H₄ loses 2e⁻ per atom (×4 = 8e⁻ lost). N in N₂O₄ gains 4e⁻ per atom (×2 = 8e⁻ gained). Balanced ✓
Question 9c
It has been suggested this reaction is a very 'environmentally friendly' source of energy. Explain this claim.
Model Answer
The only products are nitrogen gas (N2) and water (H2O) — both of which are non-toxic and non-polluting. N2 makes up ~78% of the atmosphere and is inert, so releasing more of it has no significant environmental impact.

Unlike fossil fuels, there are no CO2 or other greenhouse gases produced, and no toxic by-products such as nitrogen oxides or soot particulates.

The reaction also releases a large amount of energy per unit mass (used as rocket propellant, e.g. in the Apollo lunar module).

Counterpoint: N2H4 itself is toxic and carcinogenic — so production, storage and handling create environmental risks not reflected in the products alone.

Key Reminders

Identifying redox
Check whether any element changes its oxidation number. If none change — it is NOT redox.
Oxidising agent
Causes another to be oxidised. To do so, it must itself be reduced (ON decreases).
Reducing agent
Causes another to be reduced. To do so, it must itself be oxidised (ON increases).
Disproportionation
The same species is simultaneously oxidised AND reduced. e.g. 2Cu+ → Cu2+ + Cu
