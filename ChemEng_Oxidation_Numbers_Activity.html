Oxidation Numbers

Section 1 — The Rules

The Six Rules

These six rules let you work out the oxidation number of any element in any compound.

Rule 1
Pure elements = 0
Any uncombined element — whether a single atom or a molecule — has oxidation number zero.
Na = 0   O₂ = 0   Fe = 0   Cl₂ = 0
Rule 2
Simple ions = their charge
A monoatomic ion's oxidation number equals its ionic charge.
Na⁺ = +1   Cl⁻ = −1   Fe³⁺ = +3
Rule 3
Oxygen = −2 (usually)
Oxygen is almost always −2. Exceptions: peroxides (−1) and in OF₂ (+2).
In H₂O: O = −2   In H₂O₂: O = −1
Rule 4
Hydrogen = +1 (usually)
Hydrogen is +1 in most compounds. Exception: metal hydrides (e.g., NaH), where H = −1.
In HCl: H = +1   In NaH: H = −1
Rule 5
Fluorine = always −1
Fluorine is the most electronegative element — it is always −1 in every compound, no exceptions.
In NaF: F = −1   In OF₂: F = −1
Rule 6 ⭐
The sum rule
In a neutral compound all oxidation numbers add up to 0. In a polyatomic ion they add up to the ion's charge.
H₂O: 2(+1) + (−2) = 0   SO₄²⁻: sum = −2

💡 Tip: Rule 6 is the key calculation tool. Once you know the oxidation numbers of the other elements (from Rules 1–5), you use Rule 6 to find the unknown one.

Section 2 — Binary Compounds

Binary Compounds

A binary compound contains exactly two elements. We apply the rules in order to find the unknown oxidation number.

Worked Example 1
NaCl
1
Na is a Group 1 metal → always forms Na⁺  →  Na = +1
2
Compound is neutral → sum of oxidation numbers = 0  (Rule 6)
3
(+1) + Cl = 0  →  Cl = −1
Na = +1    Cl = −1
Worked Example 2
Fe2O3
1
O follows Rule 3 → O = −2 (no peroxide here)
2
There are 3 oxygen atoms: 3 × (−2) = −6
3
Compound is neutral → sum = 0  (Rule 6)
4
2 × Fe + (−6) = 0  →  2 × Fe = +6  →  Fe = +3
Fe = +3    O = −2
Worked Example 3
PCl5
1
Cl is a halogen (below F in Group 7) → usually −1
2
There are 5 chlorine atoms: 5 × (−1) = −5
3
Compound is neutral → P + (−5) = 0  →  P = +5
P = +5    Cl = −1

Notice how the element we're looking for is always found using Rule 6 — the sum rule. Practice using this approach now.

Section 3 — Oxoanions

Oxoanions

An oxoanion is a negative ion containing oxygen bonded to another element. The key difference from binary compounds: the ion has a charge, which becomes your target sum in Rule 6.

Worked Example 1 — Sulfate ion
SO42−
1
O = −2 (Rule 3, no exceptions here)
2
There are 4 oxygen atoms: 4 × (−2) = −8
3
The ion has charge 2− → sum of oxidation numbers = −2  (Rule 6)
4
S + (−8) = −2  →  S = −2 + 8  →  S = +6
S = +6    O = −2
Worked Example 2 — Dichromate ion
Cr2O72−
1
O = −2
2
7 oxygen atoms: 7 × (−2) = −14
3
Ion charge is 2− → sum = −2
4
2 × Cr + (−14) = −2  →  2 × Cr = +12  →  Cr = +6
Cr = +6    O = −2
Worked Example 3 — Nitrate ion
NO3−
1
O = −2
2
3 oxygen atoms: 3 × (−2) = −6
3
Ion charge is 1− → sum = −1
4
N + (−6) = −1  →  N = −1 + 6  →  N = +5
N = +5    O = −2

The method is always the same: O = −2, count the oxygens, set the sum equal to the ion charge, solve for the unknown.

Section 4 — Changes in Reactions

Oxidation & Reduction

When atoms react, their oxidation numbers can change. These changes tell us whether an atom has been oxidised or reduced.

🔺 Oxidation

An atom is oxidised when its oxidation number increases.

Think of it as "losing electrons" — the atom becomes more positive.

🔻 Reduction

An atom is reduced when its oxidation number decreases.

Think of it as "gaining electrons" — the atom becomes more negative (or less positive).

Memory aid
OIL
Oxidation Is Loss (of electrons)
RIG
Reduction Is Gain (of electrons)

Worked Example 1

Zinc reacts with copper sulfate solution. Which atom is oxidised and which is reduced?

Zn + CuSO₄ → ZnSO₄ + Cu
1
Zn on the left is a pure element → ON = 0
Zn in ZnSO₄ on the right is Zn²⁺ → ON = +2
2
Zn: 0 → +2   ON increases → Zn is oxidised
3
Cu in CuSO₄ is Cu²⁺ → ON = +2
Cu on the right is a pure element → ON = 0
4
Cu: +2 → 0   ON decreases → Cu is reduced
Zn is oxidised (+2)  |  Cu is reduced (−2 change)

Worked Example 2

Magnesium reacts with hydrochloric acid.

Mg + 2HCl → MgCl₂ + H₂
1
Mg (left): pure element → ON = 0
Mg in MgCl₂: Mg²⁺ → ON = +2  → ON increases → Mg oxidised
2
H in HCl: Rule 4 → ON = +1
H in H₂: pure element → ON = 0  → ON decreases → H reduced
3
Cl in HCl: halogen → −1. Cl in MgCl₂: still −1. No change — Cl is neither oxidised nor reduced.
Mg oxidised (+2)  |  H reduced (−1 change)

💡 Important: In every redox reaction, oxidation and reduction always occur together. You cannot have one without the other.

Section 5 — Oxidising & Reducing Agents

Oxidising & Reducing Agents

Once you know which atom is oxidised and which is reduced, you can identify the agent responsible — the species doing the oxidising or the reducing.

Oxidising Agent

The species that causes another to be oxidised.

It does this by accepting electrons — so the oxidising agent itself gets reduced.

→ Find the atom that is reduced and name its compound.

Reducing Agent

The species that causes another to be reduced.

It does this by donating electrons — so the reducing agent itself gets oxidised.

→ Find the atom that is oxidised and name its compound.

The oxidising agent is itself reduced.  ·  The reducing agent is itself oxidised.

Worked Example — Zn + CuSO₄ → ZnSO₄ + Cu
1
Zn: 0 → +2 (oxidised) — so Zn is the reducing agent (it reduces Cu²⁺ to Cu)
2
Cu: +2 → 0 (reduced) — so CuSO₄ is the oxidising agent (it oxidises Zn to Zn²⁺)
Oxidising agent: CuSO₄  |  Reducing agent: Zn
Worked Example — 2Na + Cl₂ → 2NaCl
1
Na: 0 → +1 (oxidised) — Na is the reducing agent
2
Cl in Cl₂: 0 → Cl in NaCl: −1 (reduced) — Cl₂ is the oxidising agent
Oxidising agent: Cl₂  |  Reducing agent: Na

Notice the pattern: the oxidising agent contains the atom that gets reduced; the reducing agent contains the atom that gets oxidised. The key is to find the changes first, then name the compound.
