Learn the Complete Periodic Table!

Built for advanced students: this game runs through the entire periodic table in order — Period 1 (just H and He), Period 2 (Li–Ne), Period 3 (Na–Ar), Period 4 (K–Kr), Period 5 (Rb–Xe), Period 6 (Cs–Rn), the Lanthanide Series (Ce–Lu), Period 7 (Fr–Og), and finally the Actinide Series (Th–Lr) — 118 elements in total, one row at a time.

Use the arrow keys to steer each falling block sideways along the periodic table above, and land it in its correct box on the currently active row before it falls past it. Once every element in a stage is placed correctly, the game automatically moves on to the next stage.

Correct placements score more points the faster you find them. Wrong placements cost points and send the block back up for another try. Once you're through the first three periods, every extra stage you reach (Period 4, 5, 6, the Lanthanide Series, Period 7, the Actinide Series) adds a further 20% to your points multiplier — so the later stages are worth significantly more.

Use the switches above the board any time — even mid-game — to build your own challenge: hide the category colours for double points, hide the ghost-letter hints for triple points, or tick the combined switch for a flat four times the points. For the ultimate challenge, tick "Remove all scaffolding! The expert level!": every period except the one you're currently on is hidden completely (not even an outline), so you can't see ahead to what's coming — each stage's boxes only appear as you reach it, and turn from a bare atomic number into the element symbol once solved. It also strips away all category colours, just like "Hide colours" above. This is worth a flat six times the points, and stacks with the other switches.
