Periods 1-3 Drop

This game runs through three periods in order. First Period 1 (just H and He), then Period 2 (Li through Ne), then Period 3 (Na through Ar) — 18 elements in total, one row at a time.

Use the arrow keys to steer each falling block sideways along the periodic table above, and land it in its correct box on the currently active period's row before it falls past it. Once every element in a period is placed correctly, the game automatically moves on to the next period.

Correct placements score more points the faster you find them. Wrong placements cost points and send the block back up for another try.

Use the switches above the board any time — even mid-game — to build your own challenge: hide the category colours for double points, hide the ghost-letter hints for triple points, or tick the combined switch for a flat four times the points! For the ultimate challenge, tick "Remove all scaffolding! The expert level!": every period except the one you're currently on is hidden completely (not even an outline) — including the decorative Periods 4-7 shown below, which disappear entirely too, leaving a blank sheet apart from the row you're working on. Category colours disappear and the ghost-letter hints are gone as well — worth a flat six times the points, and stacks with the other switches.
